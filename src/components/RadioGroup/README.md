## RadioGroup

Renders a set of radio buttons

### Proptypes

|   Name   |   Type   |                 Description                 | Optional | Values |   Default   |
|----------|----------|---------------------------------------------|----------|--------|-------------|
| id       | `string` | Describes the HTML id                       |          |        | `undefined` |
| label    | `string` | Describes the label                         |          |        | `undefined` |
| options  | `[{}]`   | Describes radio inputs labels and values    |          |        | `[]`        |
| value    | `string` | Describes the selected value                |          |        | `undefined` |
| disabled | `bool`   | Describes whether should render as disabled |          |        | `false`     |
| onChange | `func`   | Describes the _onchange_ function           |          |        | `undefined` |
